## Installation

* Clone the repo then `cd ./stripe-coupon`
* Run `npm install`
* Run `cd ./backend`
* Run `npm install`
* Inside `backend` directory, Run `npm run dev` to start development server
* Go back to `stripe-coupon` directory and run `npm start` to start react server
